特殊的内置函数

JavaScript常规函数包括以下8个：

  • alert函数：显示一个警告对话框，包括一个 OK 按钮
  • confirm函数：显示一个确认对话框，包括 OK Cancel按钮，返回的是一个bollean值
  • escape函数：将字符转换成Unicode码
    • escape('Visit W3school!') //Visit%20W3school%21
  • eval函数：计算表达式的结果
  • parseFloat函数：将字符串转换成浮点数字形式
  • parseInt函数：将字符串转换成整数数字形式（可指定是几进制）
    • ['1','2','3'].map(parseInt) // [1,NaN,NaN]
  • prompt函数：显示一个输入对话框，提示等待用户输入
  • unescape函数：解码由escape函数编码的字符
    • unescape('Visit%20W3school%21') //Visit W3school!

字符串对象的定义

var s = "这是我的字符串对象";

定义了s后，就有了一个字符串对象，可以访问它的长度属性（length），这个属性不需要定义，它是一个内置的属性。

console.log(s.length); //9
